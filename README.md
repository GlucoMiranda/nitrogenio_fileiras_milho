# 🌱 Nitrogenio × Arranjo de Fileiras em Milho

## 📌 Descrição

Este repositório contém a análise estatística de um experimento agronômico conduzido em delineamento de blocos ao acaso (DBC), em esquema fatorial 2 × 4, avaliando:

- Arranjos de fileiras:
  - Linhas simples
  - Linhas duplas

- Doses de nitrogênio:
  - 30, 60, 90 e 120 kg ha⁻¹

O objetivo do estudo é avaliar o efeito desses fatores sobre a produtividade do milho e modelar a resposta produtiva às doses de nitrogênio.
